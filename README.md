# NVIDIA: Fundamentals of Deep Learning: Coursera Completion

This repository contains proof of completion for the NVIDIA Deep Learning course assigned as part of UCS761 Deep Learning coursework.

---

## Course Details

Name: NVIDIA: Fundamentals of Deep Learning
Platform: Coursera
Offered by: Whizlabs (NVIDIA Content)
Link: https://www.coursera.org/learn/fundamentals-of-deep-learning

---

## Modules Completed

Module 1: Foundations of Deep Learning
Covers neuron data processing, gradient descent, perceptron training, forward and backward propagation, and activation functions.

Module 2: Advanced Deep Learning Techniques
Covers multi-class classification on MNIST, Convolutional Neural Networks, and Transfer Learning with pre-trained models.

Both modules include video lectures and graded quizzes, all of which have been completed.
